# trigger-analyses

Analysis workflows for TRIGGER data using **R**.

This repository is organized into three main folders:

## `R/`

Contains shared R scripts and utility functions used across the analyses, including functions to read TRIGGER tables, prepare data, define constants, and support the analysis workflows.

## `notebooks/`

Contains the Quarto analysis workflows and the rendered HTML reports. Each folder corresponds to a specific analysis topic.

## `outputs/`

Contains plots, tables, and other results generated by the notebooks, organized by analysis topic.

## Workflow

The general workflow is: shared functions in `R/`, analysis reports in `notebooks/`, and generated results in `outputs/`.
